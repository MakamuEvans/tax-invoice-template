tax-invoice-template
====================
Tax invoice template sample for India.

Installation
============

* Needs php5, dompdf in ./ 
* dompdf needs to be configured with DOMPDF_ENABLE_REMOTE to true

Usage
=====

From the shell, invoke it as "php pdfCreator.php"
